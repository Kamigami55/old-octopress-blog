Snack - 貪食蛇，吃掉那些點點吧！

Github

Live Demo

今天比較有閒有空，把之前一直想做的貪食蛇遊戲做出來了，使用Processing.js。

規則如同經典的貪食蛇，目標是不斷的吃點點，讓自己越長越長，但如果撞到自己就輸了。

使用方向鍵來移動，Enter鍵來開始/暫停。

我正在努力學習如何組織自己的程式碼和適當地下註解，非常歡迎指教。
